🤖 AI 상담 설정

본인의 Anthropic API 키를 입력하면 사주 상담소 답변과 종합운세 심층 분석을 Claude AI가 전문가 수준으로 작성합니다. 키가 없으면 기본(규칙 기반) 풀이로 동작합니다.

키는 이 브라우저의 localStorage에만 저장되며, Anthropic API 서버 외의 어디로도 전송되지 않습니다. 키 발급: console.anthropic.com → API Keys. 질문 1건당 대략 수십 원 수준의 API 요금이 발생합니다(모델에 따라 다름). 공용 컴퓨터에서는 사용 후 "키 삭제"를 눌러주세요.
