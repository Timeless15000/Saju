# 사주팔자 만세력 (saju.html)

생년월일시(양력)를 입력하면 사주 원국 · 오행 분포 · 십신 · 일간 성격 해석 · 대운을 보여주는
**단일 HTML 파일 웹앱**입니다. `saju.html`을 더블클릭하면 브라우저에서 바로 실행됩니다.
외부 라이브러리·인터넷 연결 불필요.

## 기능

- **사주 원국**: 년주·월주·일주·시주 8글자 (한자+한글, 오행별 색상, 십신·지장간 표시)
- **오행 분포**: 지장간 비중을 반영한 목·화·토·금·수 막대 차트 + 과다/부족 풀이
- **십신 분석**: 비겁·식상·재성·관성·인성 5그룹 집계 + 발달 그룹 해석
- **신살(神煞) · 12운성**: 천을귀인·문창귀인·도화·역마·화개·양인·백호·괴강·공망 자동 검출(내 사주에 있는 것만 표시) + 일간 기준 4주 12운성 표
- **오늘의 운세(일진)**: 오늘 날짜의 간지 vs 내 일간 십신 + 개운 오행·일지 합충으로 별점(★1~5)과 하루 조언 생성 — 매일 자정 갱신
- **일간 성격**: 갑목~계수 10가지 일간별 해석 + 신강/신약 판정
- **종합 운세**: 재물운 · 건강운(오행-오장 대응) · 직업적성 · 애정/대인관계(배우자궁) · 조심할 점 · 개운 포인트(보완 오행, 색상/방위/숫자)
- **대운**: 순행/역행·대운수 계산, 10년 단위 8개 대운, 현재 대운 강조
- **세운(연운)**: 대운 카드 클릭 → 해당 10년의 대운 테마 + 연도별 간지·나이·십신·한 줄 운세 (돕는 해 🍀 / 기복 주의 ⚠️ 표시, 올해 강조)
- **사주 상담소**: 자유 질문 상담. 기본은 규칙 기반(오프라인) — 질문에서 주제·시기를 인식해 세운 십신을 근거로 답변. **AI 모드**를 켜면 Claude가 사주 전체 데이터와 대화 이력을 근거로 전문가 수준 답변을 스트리밍으로 생성
- **🤖 AI 연동 (선택)**: 푸터 "AI 상담 설정"에 본인 Anthropic API 키 입력 → 브라우저에서 Claude API 직접 호출(서버 없음, 키는 localStorage에만 저장). 모델 선택 가능(Opus 4.8 기본 / Sonnet 5 / Haiku 4.5). 종합운세 아래 "AI 전문가 심층 분석"도 자동 생성되며 같은 사주는 캐시되어 재조회 시 비용이 들지 않음. 키가 없거나 API 오류 시 규칙 기반으로 자동 폴백
- **출생지 입력**: 도시명 자동완성(한국 25곳 + 해외 47곳, 호주 주요 도시 포함). 진태양시 보정이 출생지 경도 기준으로 자동 계산됨 — 서울 −32분, 부산 −24분, 시드니 +5분 등. 시간대 변환은 브라우저 내장 Intl 사용(1954~61 한국 UTC+8:30, 서머타임 자동 처리). 사주는 태어난 곳 기준이며 현재 거주지는 원국에 영향 없음
- **MBTI 성격유형 테스트**: 12문항(축별 3문항) 자가진단 — 이름 입력 필수, 16유형 해석 제공. 사주를 먼저 조회했다면 **일간(日干) × MBTI 결합 해석**(신강약↔E/I, 십신↔N/S·J/P, 일간 오행↔T/F)이 자동으로 붙음. 결과는 조회 기록에 저장(같은 이름의 사주 기록이 있으면 그 행에 합쳐짐)
- **조회 기록**: 이름(선택)+생년월일시+성별+출생지+MBTI를 localStorage에 자동 저장 (브라우저별). 푸터 "조회 기록 보기"에서 열람·재조회(행 클릭)·CSV 다운로드·삭제
- 옵션: 태어난 시간 모름(3주 계산), 진태양시 보정(출생지 경도 기준, 기본 꺼짐)
- 디자인: 포스텔러 등 주요 만세력 서비스 참고 — 보라 그라데이션 히어로, 카드형 레이아웃, 스티키 섹션 내비, Pretendard 폰트(CDN, 오프라인 시 시스템 폰트로 대체)

## 계산 기준

| 항목 | 기준 |
|---|---|
| 년주 | 입춘 절입 시각 기준 (태양 황경 315°) |
| 월주 | 12절기 = 태양 황경 30° 구간 (Meeus 근사식, 오차 ±수 분) + 오호둔 |
| 일주 | 자정 기준 날짜, 율리우스일(JDN+49) mod 60 |
| 시주 | 자시 23:00 시작, 2시간 간격 + 오서둔 |
| 진태양시 | 출생지 경도×4분 − 표준시 오프셋(분). 오프셋은 Intl 시간대 데이터로 출생 시점 기준 산출 |
| 대운수 | 절기 경계까지 일수 ÷ 3 (반올림), 양남음녀 순행 |
| 십신(지지) | 지장간 본기 기준 |
| 신살 | 천을·문창·양인은 일간 기준, 도화·역마·화개는 년지+일지 삼합 기준, 공망은 일주 순중공망 |
| 12운성 | 일간 기준, 양간 순행·음간 역행 |
| 일진 | 오늘 날짜(기기 시간)의 일주 간지 vs 일간 십신 |

## 개발/검증

- `dev/engine.js` — 계산 엔진 원본 (saju.html에 동일 코드가 내장됨)
- `dev/test.js` — 만세력 대조 테스트 44건 (일주 기준점, 입춘 경계, 월주 절기, 시주, 십신, 대운, 공망, 12운성, 신살)
- `dev/test-browser.html` — Node가 없는 환경에서 같은 테스트를 브라우저로 실행 (더블클릭 → 탭 제목이 TESTS-PASS면 통과)

```
cd dev
node test.js   # 44 통과 확인 (또는 test-browser.html을 브라우저로 열기)
```

엔진을 수정할 때는 `dev/engine.js`에서 고치고 테스트를 돌린 뒤, `saju.html`의
`<script>` 내 동일 함수에 반영하세요.

## 참고

- 절기 시각은 천문 근사식이므로 절기 경계(입춘 등) 수 분 이내 출생은 만세력과 대조 권장
- 1954~1961년 한국은 UTC+8:30을 사용했으므로 해당 기간 출생은 시각이 30분가량 다를 수 있음
- 야자시/조자시 구분 없이 자정에 날짜가 바뀌는 방식을 사용
